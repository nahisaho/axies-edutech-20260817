# DEMO 1 — 使用プロンプト全記録

すべての Python コードは Copilot CLI に**自然言語で依頼**した結果として Copilot が生成・実行したものです。
利用者は Python を一切手書きしていません。

## プロンプト 1: 分析前チェックリストの Markdown 化

```
Notebook の冒頭に、以下の 4 項目を Markdown セルで書いてください:
- 問い: 日本の 18 歳人口は 1990-2050 でどう推移したか？
- 予想: 全期間で単調減少しているだろう
- 反証条件: もし 2020 年以降が横ばい以上なら予想は誤り
- 出典: 文科省 学校基本調査 / 社人研 令和 5 年推計 / 私学事業団 令和 7 年度志願動向
```

**期待挙動**: Copilot が Jupyter MCP を経由し、`notebook.ipynb` の cell 0 に Markdown セルを挿入。

## プロンプト 2: データ準備とグラフ生成

```
1990〜2050 年の日本の 18 歳人口を、実測値と推計値を分けて折れ線グラフで表示してください。
条件:
- 実測は 1990-2025 (文科省学校基本調査、私学事業団 R7)
- 推計は 2030 以降 (社人研 令和 5 年中位推計)
- 2024→2025 の一時的な微増を注釈で明示
- 2040 年の 82 万人 (2005 年の約 60%) を注釈で強調
- グラフ画像は reexp/assets/demo01_18sai_jinkou.png に保存
- 出典を図の下部に記載
```

**中立性の担保**: 「減少を可視化して」のような結論誘導フレーズを避け、「どう推移したか」という中立な問いを使用。

## プロンプト 3: 予想との差分レポート

```
上記データについて、以下の観点でレポートを出力してください:
- 予想通りだった点
- 予想外だった点 (数値ベースで)
- 反証条件が満たされたかどうか
```

**期待挙動**: `print()` で構造化レポートを出力。
