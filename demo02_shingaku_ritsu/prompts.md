# DEMO 2 — 使用プロンプト全記録

## プロンプト 1: チェックリスト Markdown 化

```
Notebook の冒頭に、以下の 4 項目を Markdown セルで書いてください:
- 問い: 18 歳人口が減っているのに大学入学者数は減っていない。進学率上昇でどこまで相殺されたか？
- 予想: 進学率は 2010 年頃に 50% 前後でプラトーしていると思っていた
- 反証条件: もし 2015 → 2024 で +2pt 以上の上昇が続いていたら、プラトー予想は誤り
- 出典: 文科省 学校基本調査 令和 6 年確定値
```

## プロンプト 2: データ準備

```
1990〜2024 年 (5 年刻み + 2024) の日本の「18 歳人口 (万人)」「大学入学者数 (万人)」
「大学進学率 (%)」を DataFrame にしてください。出典は文科省 学校基本調査 令和 6 年確定値。
```

## プロンプト 3: 2 軸グラフ生成

```
上記データを 2 軸の折れ線グラフで表示してください。
条件:
- 左軸: 18 歳人口 (青、○) と 大学入学者数 (緑、■) の 2 系列 (単位: 万人)
- 右軸: 大学進学率 (赤、◇、破線) (単位: %)
- 「2015→2024 で +7.6pt」の反証注釈を追加
- タイトル「18 歳人口減 vs 大学入学者数横ばい — 進学率上昇で相殺」
- 出典: 文科省 学校基本調査 令和 6 年確定値
- グラフ画像は ../assets/demo02_shingaku_ritsu.png に保存
```

## プロンプト 4: 差分レポート

```
以下の観点で差分レポートを出力してください:
- 予想通りだった点 (2010→2015 のプラトー感)
- 予想外だった点 (2015→2024 の +7.6pt = 反証条件の 3.8 倍)
- 入学者数の維持メカニズム (18 歳人口 -47% を進学率上昇でほぼ完全相殺)
```
