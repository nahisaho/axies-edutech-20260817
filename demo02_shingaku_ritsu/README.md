# DEMO 2 — 進学率で相殺されたか？

## 概要

18 歳人口が減っているのに大学入学者数はほぼ横ばい。この "相殺" のメカニズムを、
進学率を右軸に置いた **2 軸グラフ** で可視化する。

## 予想と反証条件

- 予想: 進学率は 2010 年頃に 50% 前後で **プラトー** している
- 反証条件: 2015 → 2024 で **+2pt 以上** の上昇があれば予想は誤り

## 結果 (要約)

- 実測: 2010=50.9% / 2015=51.5% / **2024=59.1%**
- 2015 → 2024: **+7.6pt** (反証条件 +2pt の **3.8 倍**)
- → プラトー予想は **反証** された
