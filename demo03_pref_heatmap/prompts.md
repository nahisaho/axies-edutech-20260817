# DEMO 3 — 使用プロンプト全記録

## プロンプト 1: チェックリスト

```
Notebook の冒頭に、以下の 4 項目を Markdown セルで書いてください:
- 問い: 全国平均で語れない地域格差はどこにどれくらいあるか？
- 予想: 地方の小規模私大は深刻、都心 (東京・愛知・沖縄) は比較的堅調
- 反証条件: 全都道府県が均一に苦しんでいれば "地域格差" の枠組みは誤り
- 出典: 私学事業団 2025 志願動向 (デモ用近似値)

また、「本 Notebook で使用する充足率はデモ用近似値です」という注意書きも
Markdown セルに加えてください。
```

## プロンプト 2: 都道府県データ準備

```
47 都道府県の私立大定員充足率 (%) を DataFrame にしてください。
条件:
- 東京・愛知・沖縄 = 100% 以上 (定員以上に集まる)
- 東北 (秋田・青森・岩手・山形) と 山陰 (鳥取・島根) = 80% 前後
- 四国 (徳島・香川・愛媛・高知) = 80-90%
- その他は 90-100% 前後
- 数値は講演デモ用近似値であることを明記
```

## プロンプト 3: 日本地図ヒートマップ生成

```

条件:

- タイトル「47 都道府県 私立大学 定員充足率 (デモ用近似値)」

```

## プロンプト 4: 差分レポート

