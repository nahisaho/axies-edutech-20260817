# DEMO 3 — 47 都道府県 定員充足率ヒートマップ（**合成データによる可視化デモ**）

> ⚠️ **重要な位置づけ**: 本デモは **仮説検証ではなく**、japanmap を使った
> 「47 都道府県ヒートマップ描画機能」の実演です。使用データは講演デモ用の合成値であり、
> 数値は事前に「東京・愛知・沖縄が高、東北・四国・山陰が低」となるように指定して生成しています。
> したがって「集中化と過疎化」は本デモから **証拠として導出できません**。
> 実際の地域分析には一次資料（私学事業団年報・e-Stat）を必ずご参照ください。

## 概要

「Copilot CLI に日本地図での可視化を依頼すれば、japanmap パッケージのインストールから
色分けまで対話だけで完了する」を実演する。地図表現の技術デモとして位置づける。

## 予想と反証条件（本デモの位置づけ）

- 本 Notebook の分析前チェックリスト自体は「地域格差の探索」という**架空のシナリオ**として書いている
- 実際の充足率値は事前に指定した合成値のため、その"検証"はトートロジーになる
- **本デモの正しい学び**: 「地図可視化がノーコードで数分で作れること」
- **危険性の学び**: 「AI が指定した通りの数値を素直に描画するため、入力データの妥当性検証は人間が行う必要がある」

## 結果（合成データの集計）

- 都心 3 県 (東京・愛知・沖縄) 平均: 105.3%（合成値）
- 地方 (東北・四国・山陰) 平均: 82.5%（合成値）
- ※ これらは**入力プロンプトで指定した分布**の再表示に過ぎない

## ⚠️ データの注意

充足率数値は **講演デモ用の近似値** です。政策判断には必ず一次資料
（[私学事業団](https://www.shigaku.go.jp/s_center_d_shigandoukou.htm) / e-Stat）
を参照してください。
